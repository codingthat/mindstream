# Mindstream

Task management app that uses todo.txt format.

## Development

Prerequisites:

* Node.js & NPM
* [NativeScript CLI](https://docs.nativescript.org/angular/start/quick-setup#step-2-install-the-nativescript-cli) 6.0

Install required packages:

```
npm install
```

### Mobile

```
npm run android
```

### Testing

```
npm run lint
npm run test
```
